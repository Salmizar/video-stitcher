## Run Locally

Install dependencies with `npm install` (or `pnpm install` or `yarn`)

start a development server:

```bash
npm run dev
# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

Production Build:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## Running lint

```bash
npm run format
npm run lint
```
